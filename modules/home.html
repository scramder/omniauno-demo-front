Acerca de este Demo

Este es un demo frontend de Omnia Uno, un sistema de gestión empresarial completo. Esta versión utiliza datos estáticos para demostrar el diseño visual y la experiencia de usuario del sistema real.

Características del demo: Navegación SPA, diseño responsive, sistema de traducciones (ES/EN/PT), y réplica fiel de la interfaz del sistema productivo.
